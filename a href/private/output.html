In html, atributul href numeste legatura catre o alta pagina web, adica locul unde este dus utilizatorul cand executa click pe un link. Acest atribut se aplica elementului html <a>.

Linkurile pot fi de trei tipuri:

  • interne - catre locuri specifice din pagina
  • locale - catre alte pagini dar pe acelasi domeniu
  • globale - catre alte domeni in afara site-ului

HTML - Link-uri

Pentru a seta inceputul si sfarsitul unei link se poate folosi <a href="..."></a>. Un exemplu simplu ar fi:

<a href="http://www.tutorialehtml.com" target="_blank" title="Tutoriale HTML">Tutoriale HTML</a>

HTML - Ancore

Ancora este folosita pentru a lega doua sectiuni ale aceleiasi pagini. Pentru a realiza acest lucru, sectiunile trebuie sa fie denumite. Exemplu:

<h1>HTML - Hypertext Reference / href<a name="top"></a></h1>
<h2>HTML - Llink-uri text<a name="text"></a></h2>
<h2>HTML - Link-uri de e-mail<a name="email"></a> </h2>

Apoi, putem crea o referinta la aceste ancore astfel:

<a href="#top">Mergi la inceput</a>
<a href="#text">Invata despre link-uri text</a> 
<a href="#email">Invata despre adrese de e-mail</a>

HTML - Link-uri de download

Forma unui link de download este exact aceeasi ca a unui link normal de text.

<a href="http://www.tutorialehtml.com/htmlT/text.zip" title="Trimite nelamuriri">Text Document</a>
